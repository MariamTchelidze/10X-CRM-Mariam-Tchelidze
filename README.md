# 📚 10X CRM

> *A modern educational CRM system built with Vanilla JavaScript.*

---

## ✨ About

10X CRM is a modern educational Customer Relationship Management (CRM) system designed to manage students, courses, tasks, schedules, and daily workflow in a clean and user-friendly environment.

Unlike a traditional CRUD project, this application focuses on delivering a polished user experience, intuitive navigation, responsive layouts, and modern UI principles while demonstrating core JavaScript concepts.

This project is being developed as a JavaScript Final Exam Project and serves as both a technical challenge and a personal portfolio piece.

---

## 🎯 Vision

The goal of this project is to build an educational CRM platform that feels modern, elegant, and enjoyable to use.

Key design principles include:

- Minimalistic Interface
- User-Centered Experience
- Responsive Design
- Light & Dark Themes
- Smooth Animations
- Clean Information Architecture
- Reusable Components
- Maintainable Code Structure
